<h3>How to make it work?</h3>

1.  Clone the repo:
        
         git clone https://github.com/AllaOz/React-project.git
2.  Go to the project folder:

        cd my-app
3.  Install dependencies using:

       ```
        npm install
       ```
4.  Run:
       ```
        npm start
       ```
